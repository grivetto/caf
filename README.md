# CAF Italia

Sito web professionale per un Centro di Assistenza Fiscale (CAF), realizzato con HTML, CSS e JavaScript puri.

## Caratteristiche

- Design responsive (mobile-first)
- Tema personalizzato con palette blu scuro e arancione
- Sezioni: servizi fiscali, chi siamo, novità, download moduli, contatti
- Statistiche animate
- Menu mobile hamburger
- Back-to-top button
- Form contatto integrato

## Struttura

```
caf/
├── index.html          # Pagina principale
├── css/
│   └── style.css       # Stili completi
├── js/
│   └── main.js         # Interattività (menu, contatori, form)
└── img/                # Risorse immagini
```
